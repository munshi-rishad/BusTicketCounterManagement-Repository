# Bus Ticket Counter Management System

This project is a complete bus ticket booking and management system. It allows for efficient handling of ticket bookings, customer information, and payment details. The system is built using Java and Oracle Database, ensuring robust performance.

## Features

- Bus ticket booking and cancellation
- Customer management (details, profiles)
- Payment processing integration
- Route and schedule management
- Data stored in Oracle database

## How to Use

1. Clone the repository to your local machine.
2. Set up the Oracle database and import the provided SQL scripts.
3. Configure your database connection in the application.
4. Run the project using your IDE (e.g., Eclipse or IntelliJ for Java).

## Project Files

- Core project code in Java
- SQL scripts for database structure
- Documentation (Word file)
